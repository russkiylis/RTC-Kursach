# Сделать курсовую по РТЦиС с минимальными усилиями
Эта программа для MATLAB поможет воплотить это в жизнь!

## Краткий гайд
### Методы вывода
- `showSignal()` - выводит графики сгенерированных сигналов по информации из "дано"
- `showNoiseSPM()` - выводит график СПМ шума, значение *W0* и *ωгр* по информации из "дано"
