# Сделать курсовую по РТЦиС с минимальными усилиями
Эта программа для MATLAB поможет вам в данном вопросе.

## Краткий гайд
### Методы вывода
- `dano_showSignal()` - выводит графики сгенерированных сигналов по информации из "дано"
- `dano_showNoiseSPM()` - выводит график СПМ шума, значение *W0* и *ωгр* по информации из "дано"
- `p1_showSpectrFFT()` - выводит спектры сигналов, посчитанный через БПФ. Также показывает выбранный сигнал
